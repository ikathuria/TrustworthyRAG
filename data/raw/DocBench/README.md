# DocBench Dataset
We used the DocBench dataset for our experiments. Below is a brief description of the dataset:
The DocBench dataset is a collection of documents designed for benchmarking document retrieval systems. It includes a diverse set of documents from various domains, making it suitable for evaluating the performance of retrieval models across different types of content.

The dataset is organized into several categories, each representing a different type of document. These categories include scientific articles, legal documents, technical manuals, and more. Each document is accompanied by metadata that provides additional context, such as the document's title, author, publication date, and keywords.

To use the DocBench dataset, users can download it from the [official repository or website](https://github.com/Anni-Zou/DocBench/).
